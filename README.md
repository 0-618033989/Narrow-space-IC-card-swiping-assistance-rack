# Narrow-space-IC-card-swiping-assistance-rack
这个小东西的设计使用场景是拥挤的公交车内，方便身处刷卡机背面的乘客刷卡。

整个装配体需要4种6个零件，分别为L型杆（2个）、I型架（1个），M5*25的螺栓（或者自攻螺钉，2个），适配你的IC卡的自封袋（比如尺寸为60\*80，1个）。

在这个仓库中存有L型杆、I型架的STL\STEP\SLDPRT文件，以及不含连接件的装配体的STEP\SLDASM文件，可以使用SW2025及以上版本的同类软件打开。使用者可以下载相关文件，通过某宝等平台的3D打印服务获取实体产品。

请勿将本产品用于北京航空航天大学
